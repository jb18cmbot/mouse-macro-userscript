# 🖱️ 鼠标侧键宏 - Tampermonkey 用户脚本

## 功能简介

这是一个 Tampermonkey 用户脚本，可以将鼠标侧键（Button 3/4/5）映射为键盘按键或组合键，支持自由添加多个映射。

## 主要特性

- ✅ 支持鼠标侧键（Button 3/4/5）映射
- ✅ 可映射为单个字母键或 `字母 + Space` 组合键
- ✅ 完全屏蔽鼠标侧键的默认行为（如浏览器前进/后退）
- ✅ 图形化配置界面，无需修改代码
- ✅ 支持添加/删除多个映射
- ✅ 内置按键测试工具
- ✅ 配置持久化保存

## 安装方法

1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 浏览器扩展
2. 点击 Tampermonkey 图标 → 创建新脚本
3. 复制完整脚本代码粘贴进去
4. 保存（Ctrl+S 或 Cmd+S）

## 使用方法

### 打开设置界面

点击 Tampermonkey 图标 → 选择 "⚙️ 鼠标宏设置"

### 配置映射

1. **启用/禁用**：勾选 "启用鼠标宏" 复选框
2. **添加映射**：点击 "➕ 添加新映射" 按钮
3. **配置映射**：
   - 选择鼠标按键（Button 3/4/5）
   - 输入键盘按键（单个字母，如 `q`）
   - 可选：勾选 "+ Space" 实现组合键
4. **删除映射**：点击映射右侧的 "🗑️ 删除" 按钮

### 测试按键

1. 点击 "🔍 测试按键" 按钮
2. 按下鼠标或键盘，查看识别的按键信息
3. 再次点击 "⏹️ 停止测试" 关闭测试模式

### 保存配置

点击 "💾 保存" 按钮，刷新页面后生效。

## 配置示例

### 示例 1：单键映射

- **鼠标按键**：Button 3（侧键后退）
- **键盘按键**：`q`
- **效果**：按下侧键后退 = 按下 Q 键

### 示例 2：组合键映射

- **鼠标按键**：Button 4（侧键前进）
- **键盘按键**：`w`
- **勾选**：+ Space
- **效果**：按下侧键前进 = 同时按下 Space + W

### 示例 3：仅屏蔽默认行为

- **鼠标按键**：Button 3
- **键盘按键**：留空
- **效果**：侧键不触发任何操作，但屏蔽浏览器默认行为

## 技术细节

### 按键模拟机制

脚本通过 `KeyboardEvent` 模拟键盘按键，按下顺序为：

- **单键**：`keydown` → `keyup`
- **组合键**：`Space keydown` → `字母 keydown` → `字母 keyup` → `Space keyup`

### 事件拦截

脚本在捕获阶段（`capture: true`）拦截以下鼠标事件：

- `mousedown`
- `mouseup`
- `auxclick`（非主按钮点击）
- `click`（额外保险）

### 配置存储

配置通过 `GM_setValue` 存储在 Tampermonkey 的本地存储中，格式为：

```javascript
{
  enabled: true,
  mappings: [
    { mouseButton: 3, key: 'q', space: false },
    { mouseButton: 4, key: 'w', space: true }
  ]
}
```
